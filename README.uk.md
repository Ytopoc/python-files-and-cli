# Python - файли, парсинг, CLI

Три невеликі вправи на роботу з файлами, регулярні вирази, `pathlib` та базовий командний диспетчер.

## Стек

- Python 3.12 (тільки стандартна бібліотека)

## Вправи

### `hw4_1.py` - парсер файлу зарплат

`total_salary(path)` читає текстовий файл, де кожен рядок - `Last_Name,salary`, і повертає `(сума, середнє)` по всіх рядках. Повертає `(0, 0)` і виводить повідомлення при некоректному шляху чи погано сформованому вводі.

### `hw4_2.py` - парсер CSV з котиками

`get_cats_info(path)` читає CSV з форматом `id,name,age` по рядку і повертає список словників:

```python
[{"id": "60b90c1c13067a15887e1ae1", "name": "Tayson", "age": "3"}, ...]
```

Повертає `None` при помилках шляху чи рядках, що не мають рівно трьох полів.

### `hw4_4.py` - CLI контактів

Простий in-memory диспетчер контактів з такими командами:

| Команда | Дія |
|---------|-----|
| `add <ім'я> <телефон>` | Створити контакт |
| `change <ім'я> <телефон>` | Оновити телефон |
| `phone <ім'я>` | Показати контакт |
| `all` | Показати всі контакти |
| `hello` | Привітання |
| `close` / `exit` | Вийти |

`parse_input` розбиває рядок користувача; кожен обробник повертає рядок, який виводить головний цикл.

## Запуск

```bash
python hw4_1.py
python hw4_2.py
python hw4_4.py
```
